# Full-Stack Challenge - Frontend

## Env Requirement

```
node @ 14.19.1 + 
npm @ 6.14.16 +
Typescript @ 4.3.2 +
```

## Setup

```
yarn // recommand
npm i
```

```
yarn build // recommand
npm run build
```

```
yarn start // recommand
npm run build
```

> Please make sure to run `build` and `start` script to run the client service as the production mode, instead of running on the development mode by the `dev` script.

## Tech Stack

* `Typescript`
* `Next.js`
* `TailwindCSS`
* `Material-UI`
* `apollo/client`
* `slate`: For the customized WYSIWYG Editor.
